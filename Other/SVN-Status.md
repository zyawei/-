| Code    | Description                         |
| ------- | ----------------------------------- |
| Lock    | svn已经在.svn目录锁定了abc.         |
| Modify  | 文件内容已经在本地修改过了          |
| Modify  | 文件属性有修改，但没有内容修改      |
| X       | 这个目录是外部定义的一部分          |
| ?       | svn并没有管理这个文件               |
| !       | svn管理这个，但它可能丢失或者不完整 |
| ~       | 进行了版本控制，但类型已经改变      |
| Ignore  | svn不管理这个，配置确定要忽略它     |
| Delete  | 这个文件预定要删除                  |
| Add     | 这个文件预定要添加                  |
| Compare | 这个文件在更新时发生冲突            |
| Re..    | 这个文件预定要被替换                |
| Switch  | 这个文件已经跳转到了分支            |
